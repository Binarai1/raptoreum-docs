### Installer

1. Download the latest release [here][#setup] or on [GitHub][wallet-gh]
2. Run the installer (.exe or .dmg) and follow the steps
3. Run the RaptoreumCore Daemon (raptoreumd)
4. Open the wallet
    - **CLI**
      Run raptoreum-cli.exe from the command line to use
    - **GUI**
      Run raptoreum-qt.exe to open 
5. Allow Daemon to Sync (Use the [bootstrap package](docs/wallet/bootstrap))
