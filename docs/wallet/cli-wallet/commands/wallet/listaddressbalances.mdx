# listaddressbalances

*listaddressbalances ( minamount )*

Lists addresses of this wallet and their balances

```bash
raptoreum-cli listaddressbalances 
raptoreum-cli listaddressbalances 10
```

<ReactPlayer playing controls url={[
    {src: `${require('./assets/listaddressbalances.webm').default}`, type: 'video/webm'}]} />

## Arguments

| Argument | Type | Description |
|--|--|--|
| minamount | (numeric) | Minimum balance in RTM an address should have to be shown in the list  |

```bash
Arguments:
1. minamount               (numeric, optional, default=0) Minimum balance in RTM an address should have to be shown in the list
```

## Output

| Result | Type | Description |
|--|--|--|
| "address" | (string) | The raptoreum address and the amount in RTM |

```bash
Result:
{
  "address": amount,       (string) The raptoreum address and the amount in RTM
  ,...
}
```

## Examples

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listaddressbalances", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listaddressbalances", "params": [10] }' -H 'content-type: text/plain;' http://127.0.0.1:9998/
```
