Get started quickly with the Raptoreum Wallet CLI.

## Windows

Download the Windows [Release](https://github.com/Raptor3um/raptoreum/releases)

| Release | URL |
|--|--|
| Windows Installer | [raptoreum-1.2.15.2-win64-setup.exe](https://github.com/Raptor3um/raptoreum/releases/download/1.2.15.2/raptoreum-1.2.15.2-win64-setup.exe) |
| Windows Standalone | [raptoreum_1.2.15.2_windows_64.zip](https://github.com/Raptor3um/raptoreum/releases/download/1.2.15.2/raptoreum_1.2.15.2_windows_64.zip) |

#### Installer:

- Run the Installer
- Follow the steps
- Run Raptoreum Core
- Run CLI from `C:\Program Files\RaptoreumCore\daemon\raptoreum-cli.exe`
- Allow Wallet to Sync (Use the bootstrap)

#### Standalone Zip:

- Extract the zip
- Use the following per use case:
  * `raptoreumd` - Raptoreum Daemon
  * `raptoreum-qt` - Raptoreum Wallet GUI
  * `raptoreum-cli` - Raptoreum Wallet CLI
  * `raptoreum-tx` - Raptoreum Tx Utility
- Run Daemon and `./raptoreum-cli` for Wallet CLI
- Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))

#### Precompiled Binaries:

- Download release for Ubuntu 18 or 20:
  * `wget https://github.com/Raptor3um/raptoreum/releases/download/1.2.15.2/raptoreum_1.2.15.2_ubuntu20_64.tar.gz`
- Extract contents:
  * `tar xzvf raptoreum_1.2.15.2_ubuntu20_64.tar.gz`
- Use the following per use case:
  * `raptoreumd` - Raptoreum Daemon
  * `raptoreum-qt` - Raptoreum Wallet GUI
  * `raptoreum-cli` - Raptoreum Wallet CLI
  * `raptoreum-tx` - Raptoreum Tx Utility
- Run Daemon and `raptoreum-cli` for Wallet CLI
- Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))
