# Quick Start

Get started quickly with the Raptoreum Wallet GUI.

<!-- <ReactPlayer playing controls url={[
    {src: `${require('./assets/placeholder.webm').default}`, type: 'video/webm'}]} /> -->


## Windows

Download the Windows [Release](https://github.com/Raptor3um/raptoreum/releases)

<ReleaseWidget releaseType="windows" />

#### Installer:

- Run the Installer
- Follow the steps
- Run Raptoreum Core
- Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))

#### Standalone Zip:

- Extract the zip
- Use the following per use case:
  * `raptoreumd` - Raptoreum Daemon
  * `raptoreum-qt` - Raptoreum Wallet GUI
  * `raptoreum-cli` - Raptoreum Wallet CLI
  * `raptoreum-tx` - Raptoreum Tx Utility
- Run Daemon and `raptoreum-qt` for Wallet GUI
- Allow Wallet to Sync (Use the bootstrap)

## Unix/Linux

Download the Unix/Linux [Release](https://github.com/Raptor3um/raptoreum/releases)

<ReleaseWidget releaseType="ubuntu" />

#### Precompiled Binaries:

- Download release for Ubuntu 18 or 20:
- Extract contents:
```bash
  tar xzvf raptoreum_1.2.15.2_ubuntu20_64.tar.gz
```
- Use the following per use case:
  * `raptoreumd` - Raptoreum Daemon
  * `raptoreum-qt` - Raptoreum Wallet GUI
