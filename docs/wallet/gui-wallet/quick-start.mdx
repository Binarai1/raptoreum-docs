## Setup

### Installer

- Run the Installer (.exe, .dmg)
- Follow the steps
- Run Raptoreum Core
- Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))

### Windows Binaries

- Extract the zip
- Use the following per use case:
  
  - `raptoreumd` - Raptoreum Daemon
  - `raptoreum-qt` - Raptoreum Wallet GUI
  - `raptoreum-cli` - Raptoreum Wallet CLI
  - `raptoreum-tx` - Raptoreum Tx Utility

- Run Daemon and `raptoreum-qt` for Wallet GUI
- Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))

### Unix / Linux Binaries

1. Download release for Ubuntu 18 or 20:
2. Extract contents:

    ```bash
    tar xzvf raptoreum_1.2.15.3_ubuntu20_64.tar.gz
    ```

3. Use the following per use case:

   - `raptoreumd` - Raptoreum Daemon
   - `raptoreum-qt` - Raptoreum Wallet GUI
   - `raptoreum-cli` - Raptoreum Wallet CLI
   - `raptoreum-tx` - Raptoreum Tx Utility

4. Run Daemon and `raptoreum-qt` for Wallet GUI
5. Allow Daemon to Sync (Use the [bootstrap](/docs/wallet/bootstrap))
