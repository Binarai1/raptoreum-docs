# Requirements

Smartnodes are servers running a specially configured instance of the raptoreumd (daemon). These nodes make Raptoreum immune to reorganization, 51%, double spend type atacks. In the future they will also be used to store and execute smart contracts. This document will show you how to setup your own Smartnode with script and manually.

:::info Requirements

Smartnodes require a minimum:
- 2 CPU cores
- 2GB of RAM

on a high performance VPS or bare-metal server.
SSD / NVME powered storage is also highly recommended.

:::
